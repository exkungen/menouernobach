Over mij

Mijn naam is Menouer Nobach, ik woon in Heemskerk en ik ben 18 jaar oud. Momenteel zit ik in het 2e leerjaar op het Media College Amsterdam met als opleiding "Media Developer". Na mijn opleiding wil ik graag software-engineer studeren aan de Hogeschool van Amsterdam. Mijn interesses liggen vooral bij back-end maar ik vind front-end ook leuk om te doen. Later zou ik mijn eigen bedrijf willen beginnen, en als dat niet lukt wil ik bij een klein development bedrijf te werk gaan. Naast programmeren ben ik ook erg goed op het gebied van communicatie en samenwerking

Vaardigheden

Feedback

Ik sta altijd open voor feedback; ik ben een makkelijk aanspreekbaar persoon die het zeer op prijs stelt als mensen hun meningen laten weten over mijn projecten.

Functioneel

Een bruikbaar en goed eindproduct is uiterst belangrijk in de programmeerwereld, daarom streef ik altijd naar een top resultaat waar de klant en ik zelf tevreden mee zijn.

Creatief

Ik probeer altijd iets unieks te verzinnen als ik een website bouw of wanneer ik iets aan het programmeren ben. Ook denk ik altijd oplossinggericht en pak problemen bij de bron aan.

Web Design

Altijd op de hoogte zijn van de laatste trends is super belangrijk voor web designs, daarom besteed ik naast het programmeren ook tijd aan het bestuderen van web designs zodat ik altijd bij blijf van de nieuwste looks, functionaliteiten en gebruikers-ervaring.

Communicatie

Communicatie en samenwerking is belangrijk, niet alleen om een goed product te produceren, maar ook om elkaar niet in de weg te zitten. Op dit gebied ben ik erg goed in, ik ben naast makkelijk aanspreekbaar te zijn ook flexibel.

Web Development

Het bouwen van websites heeft te maken met het bestuderen van programmeertalen met deze talen kunt u website en applicaties bouwen. Ik beheers zelf de volgende talen: "HTML", "CSS", "PHP", "Ecma2015", "JavaScript" en "MySQL". Ook beheers ik de volgende frameworks: "NodeJS", "Bootstrap" en "SASS"

Projecten

Bekijk hier mijn laatste werk.

instaclone

Web Development - Back-end

De instaclone website was een project voor het vak "Back-end programming." Het doel was om gebruikers foto's te kunnen laten uploaden met een beschrijving erbij, De plaatjes worden geupload naar de database en op de home pagina weergeven. Het was een leuk process om dit project te maken en ik heb er veel kennis van opgestoken


Karin Oosterveer

Web Design

Voor het bedrijf Lumen werd mij gevraagd of ik een website kon maken voor "Karin Oosterveer" er werd mij een wire-frame aangeleverd waarop ik de lay-out van de website moest baseren. Het werkprocess was prima verlopen, en ik heb veel vaardigheden opgedaan in "Bootstrap 4" door deze opdracht.


Gastenboek

Web Development - Back-end

Met back-end programming heb ik geleerd hoe ik met doormiddel van PDO mijn website beter bevelig tegen exploits. Ik heb een gastenboek gemaakt waarmee gebruikers berichten kunnen plaatsen op de home-pagina. Ook is hier gebruik gemaakt van regular experessions, dit betekent dat als gebruikers een scheldwoord plaatsen dit automatisch gefilterd wordt. Alle berichten worden ook geplaatst in de database.


Web Development - AJAX Vierkant

Front-end

Doormiddel van het gebruik van AJAX, heb ik een webpagina gemaakt waarme je kleuren kan selecteren en daarmee een vierkantje kan inkleuren. Door het gebruik van AJAX hoeft hierdoor de pagina niet herladen te worden voordat je resultaat ziet.


Web development - AJAX Database

Front-end

Met het vak front-end heb ik dit project gemaakt, het hoofddoel van dit project was om kennis te maken met AJAX. AJAX is heel belangrijk in de webdevelopment wereld om websites interactief te maken. In dit project kun je kun je verschillende dingen selecteren uit een dropdown menu. Zonder dat de pagina herlaad worden er resultaten weergeven.


Web development - The Wall

Back-end

Ik in combinatie met Melissa Vriend moesten een wall maken. Dit project moest in twee-tallen zodat je leert om samen te werken en te communiceren met mede klasgenoten, want communicatie is cruciaal in het bedrijfsleven. Het doel van dit project was om plaatjes te uploaden en te weergeven op de hoofdpagina. Uiteindelijk moesten we dit project presenteren aan de ouderen van de studenten, Melissa en ik kwamen in de finale maar hadden jammer genoeg niet gewonnen. Ik vond dit een super leuk project om dat ik er veel kennis in back-end programming heb opgestoken.
